Договір щодо оренди
Це внутрішня CRM-система для громадських організацій, яка допомагає зручно, безпечно та структуровано працювати з даними людей, яким надається допомога (бенефіціарів). Система дозволяє ГО створювати проекти, керувати працівниками, вести облік запитів (лідів), заповнювати форми опитування, генерувати звіти та контролювати доступ до даних по ролям.
